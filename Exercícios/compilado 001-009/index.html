Parágrafos e quebras de linha

Para adicionar um paágrafo basta utlizar a tag <p>

Se precisar quebrar o texto, utilizar a tag <br>

Imagens e favicon

Como adicionar imagens

Usar a tag <img>. Usar o ctrl+espaço para abrir o seletor de imagens que está na pasta. Sempre colocar o alt a descrição da imagem

Logo do css

Como adicionar favicon

Antes do title da head usar a tag <link>, selecionar o icon favicon e usar o ctrl+espaço para selecionar o icon a ser utilizado

Hierarquia de títulos

Em HTML temos uma hierarquia de títulos que se expressam pelas tags h1, h2... até h6. Essas tagas não se referem ao tamanho da letra dos títulos, mas sim da sua ordem de impotância, como uma divisão entre títulos e subtítulos. Toda a estilização que se refere a forma é delegada às CSS

Formatação de textos

Semântica

A Semântica tema função ade atribuir significado
Por exemplo, a tag <adress> atribui significado àquela frase, monstrando que é um endereço

Negrito e itálico

Para usar o negrito de forma semântica usar a tag <strong>

Para dar ênfase em um trecho do texto, usar tag <em>

Dica de como envelopar trechos: crtl+shift+p. Depois enter, digitar o que quer e enter

Marca Texto

Para marcar o texto, utilizar a tag <mark>. Para mudar a cor e estilização é necessário usar as CSS

Listas OL e UL

Listas ordenadas

Usar a tag OL para esse tipo de lista em que a ordem dos itens é relevante e casa item deve ser iniciado pela tag <li>. Essa tag possui um parâmetro type que deve ser inserido dentro das chaves da OL. Além disso, caso queira que a lista começa em 5 e não em 1, por exemplo, usar o parâmtro start para indicar onde se quer iniciar. A lista UL é praticamente a mesma coisa

Exemplo:

  1. Acordar
  2. tomar banho
  3. tomar café da manhã

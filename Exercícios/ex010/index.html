Links externos

Você pode acessar o repositório público no GitHub

Pra fazer isso é necessário envelopar a frase que você quer que seja o link, e depois colar o link entre as aspas da tag <a>;

Para fazer com que esse link abra em uma nova janela colocar depois do link <target="_blank" rel="external"

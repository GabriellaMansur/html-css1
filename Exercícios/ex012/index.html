Inserindo vídeos hospedados localmente

Este vídeos está hosepdado do meu próprio servidor

Para colocar o vídeo é a mesma coisa que foto, e é importante colocar todos formatos de vídeo por conta da compatibilidade com determinados navegadores

A ordem tem total importânia nesse caso

  1. Para mudar o tanho que o vídeo vai ficar no site é só colocar width e escever um número entre as aspas

Colocar capa no video

Para colocar capa no vídeo basta escrever poster e procurar o arquivo com o crtl espaço

Inserindo vídeos do YouTube

Para fazer isso é só ir no vídeo postado no YouTube copiar o incorparar dele e colar aqui no vs code. Também dá para mudar o tanho que o vídeo vai ficar no site, apesar de já vir com um tamanho pré definido

Inserindo vídeos com o Vimeo

Também dá pra fazer a mesma coisa que o youtube, só que nesse dá para deixar os vídeos privados. Só para assinantes, por exemplo

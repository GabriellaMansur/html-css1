Reproduzindo áudios

Vamos aprender a reproduzir áudios em HTML5

Para reproduzir o ádio colocar a tag audio midia, escolher o arquivo como ctrl espaço, seguido dos parâmetros controls e autoplay, mas fazer assim tem um problema de compatibilidade

Desse segundo modo, seguir os sehuintes passos

  • Tag audio
  • Apagar tudo que estiver entre ela
  • Escolher o souuce src
  • Carregar o arquivo na pasta
  • No type colocra o tipo de arquivo.
  • Colocar na tag audio preload metadata
    1. Para mp3 utilizar audio/mpeg
    2. Para ogg, utilizar audio/ogg
    3. Para wav, utilizar audio/wav

Importante notar quea ordem é relevante nesse caso, pois ele vai tentar carregar primeiro o tipo de arquivo que estiver no topo da lista. CAso o navegador não seja compatível com nenhum dos aquivos, escrever uma mensagem e deixar um link para baixar o audio
